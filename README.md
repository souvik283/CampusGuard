# 🚨 Fake Placement Scam Detection System  

#Backend Repo: "https://github.com/Rajib-dx/campusGuard-Backend"

## 💡 Solution Overview  
We propose a **Smart Scam Detection Platform** that helps students verify job opportunities, detect scams, and build trust in the hiring process using AI and community-driven insights.

---

## 🎯 Key Features  

### 🔐 Verified Recruiter System  
- Recruiters verified using official email/domain  
- Trust score for each recruiter  

### 🧠 AI Scam Detection  
- Detect suspicious job descriptions  
- Classify offers as **Safe / Risky / Scam**  

### 🌐 Website & Link Scanner  
- Analyze job links  
- Detect fake or cloned company websites  

### 🎥 Deepfake Interview Detection *(Unique Feature)*  
- Identify fake video interviews  
- Analyze face and voice inconsistencies  

### 👥 Community Reporting  
- Students can report scams  
- Shared database of fraudulent activities  

### 📲 WhatsApp/Telegram Integration  
- Forward messages to bot for instant verification  

### 🏢 Company Verification Dashboard  
- Search companies and view:
  - Official website  
  - Verified HR contacts  
  - Scam history  

### 💸 Payment Alert System  
- Warn users if job asks for money  
- Detect keywords like “registration fee”  

### 📊 Admin Analytics Dashboard  
- Track scam trends  
- Identify high-risk areas  

### 🪪 Student Protection Mode  
- Simple checklist before applying  
- Visual indicators (✅ Safe / ❌ Risky)

## ⚙️ How It Works  

1. User enters job details or link  
2. System analyzes:
   - Recruiter authenticity  
   - Website legitimacy  
   - Content patterns  
3. AI model assigns risk score  
4. Community data cross-checks reports  
5. Final output shown as:
   - ✅ Safe  
   - ⚠️ Suspicious  
   - ❌ Scam
